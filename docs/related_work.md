# 📚 Related Work

This project is inspired by:

**"YOLO v4 and ConvLSTM based computer vision system for real-time detection of unsafe behaviours in workplace"**
Oğuzhan Önal, Emre Dandıl

## Key Idea
- YOLO used for object detection
- ConvLSTM used for temporal modeling
- Focus on real-time unsafe behavior detection in workplace environments

## Difference from our approach
- We use YOLOv8 instead of YOLOv4
- We replace ConvLSTM with 3D CNN temporal modeling
- We extend to 8-class classification system
